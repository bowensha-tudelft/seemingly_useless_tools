# seemingly_useless_tools

A small index of utilities and related files. Each item below links to its own folder for details.

## Contents

- [gjf_swap](gjf_swap/README.md) — reorder selected atoms in Gaussian `.gjf` files
- [mol_box](mol_box/README.md) — calculate simple molecular box quantities
- [deepseek-pricing-extension](deepseek-pricing-extension/README.md) — add prices to DeepSeek Usage tooltips
- [bilibili-tab-sorter](bilibili-tab-sorter/README.md) — sort open Bilibili tabs by duration

## Notes

- Python-based tools use only the Python standard library.
- Browser extensions are packaged in their own folders and can be loaded unpacked from there.
- The list can grow over time.
